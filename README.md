# QPerceptron
After reading the paper Turing, A. M. (1950). *Computing machinery and intelligence*. *Mind*, 59(236), 433–460. https://doi.org/10.1093/mind/LIX.236.433. He says that, in simple terms, that a discrete-machine cannot simulate a brain. So, the implementation of a Perceptron (one of the most simple machine learning implementation) in Qiskit (For quanthum computing) arrived to my mind.

# Landmarks
* Implement a Perceptron using Qiskit.
* Implement a Perceptron from scratch for a classical discrete-machine.
* Compare the results.
